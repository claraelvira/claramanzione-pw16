# GioSalus

Applicazione full-stack API-based sviluppata per la gestione di pazienti e appuntamenti in ambito sanitario.

Il progetto è stato realizzato come elaborato per il Project Work PW16 e simula una piattaforma digitale pensata per una clinica privata.

## Tecnologie utilizzate
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## Funzionalità
- Gestione pazienti
- Prenotazione appuntamenti
- Comunicazione tramite API REST
- Separazione tra frontend e backend

## Struttura del progetto
- `frontend/` contiene l’interfaccia web dell’applicazione
- `backend/` contiene il server Node.js e le API REST
- `README.md` contiene la documentazione del progetto

## Obiettivo
L’obiettivo del progetto è simulare una semplice applicazione sanitaria full-stack, mostrando il funzionamento della comunicazione tra interfaccia utente e backend tramite API REST.
